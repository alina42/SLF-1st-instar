The supplementary material includes:

1)Table with the unique sequences of ingested plants.
-This file contains information on each unique plant species identified and the sequence used to determine the plant species. This information includes the scientific species name along with the plant family and common name. The sequence with the highest percentage match is also included along with its length.

2)Table with raw mean quality scores.
-This file has all the mean quality scores generated by running Galaxy\'92s FastQC on the raw unique sequence abundance files.

3) Raw sequences for all samples
-This zip file contains all the raw data received after sending sample for NGS.

4)Folder containing all unique sequence abundance files.
-This folder holds all the unedited uniques sequences generated by converting the files obtained from NGS into readable .csv files.
